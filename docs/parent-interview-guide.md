# Phase 0 — Parent interview guide

Run these conversations with your parents **this week**. Take rough notes — perfect answers are not required.

## Questions (from product plan)

1. **How do you currently create papers?** (Word? Past papers? Textbook?)
2. **How long does one paper take?** (hours, including memo)
3. **What is the most frustrating part?** (finding questions, memos, moderation, formatting?)
4. **What format does your moderator require?** (cover page, mark grid, Bloom table?)
5. **How do you build memorandums today?**
6. **How do you assign Bloom's Taxonomy?** (manually, school template, rarely?)
7. **How do you ensure CAPS/IEB coverage?** (ATP, textbook, colleague?)
8. **How many papers do you create per year?** (per subject)

## Capture template

Copy this into a note after each conversation:

```
Date:
Subject(s) taught:
Grade(s):

Biggest pain point (one sentence):

Typical assessment types:
- Cycle test: yes/no, how often
- June exam: yes/no
- Practical/PAT: yes/no

Moderator must-haves:

Would use an app tomorrow if it did X:

Deal-breakers:
```

## What we do with answers

| Answer affects | Example |
|----------------|---------|
| Wizard step order | If memos are the pain, prioritize memo generation in Phase 2 |
| Default marks/time | Grade 11 cycle test often 50 marks / 60 min |
| Export format | School cover page upload priority |
| Topic lists | Validate our Maths / Life Sciences topic lists |
| Pilot grade | Weight PDF ingest toward Grade 10 vs 11 vs 12 |
